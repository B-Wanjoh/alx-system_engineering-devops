-The script inside the file "0-current_working_directory" prints the absolute path name of the current working directory.
-The script inside the file "2-bring_me_home" changes the working directory to the user’s home directory.
-The script inside the file "6-firstdirectory" creates a directory named my_first_directory in the /tmp/ directory.
-The script inside the file "10-back" changes the working directory to the previous one.
-The script inside the file "11-lists" lists all files (even ones with names beginning with a period character, which are normally hidden) in the current directory and the parent of the working directory and the /boot directory (in this order), in long format.
-The script inside the file "12-file_type" prints the type of the file named iamafile.
-The script inside the file "14-copy_html" copies all the HTML files from the current working directory to the parent of the working directory.
